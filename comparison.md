# Zero-shot vs Few-shot Prompting

| Feature | Zero-shot | Few-shot |
|----------|-----------|-----------|
| Examples | No | Yes |
| Accuracy | Good | Better |
| Consistency | Medium | High |
| Prompt Length | Short | Longer |
| Token Usage | Low | Higher |
| Best For | Simple tasks | Complex extraction |

## Observation

Zero-shot prompting performed well on structured text but occasionally struggled with unusual formatting.

Few-shot prompting produced more consistent JSON and handled diverse text formats more effectively.
